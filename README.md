# lemona 반응형 웹 홈페이지

## 개요
  (여기에 작품에 대한 소개를 작성)
  가변 Full Screen Slide 유형의 홈페이지

## 작업내용
0. 시안 및 프로토타입 제작(figma)
1. 헤더 toggle gnb 반응형으로 구현
2. main visual 영역 애니메이션
3. media query와 flex layout 활용
4. 동영상 모달창 구현
5. smooth page scroll


## 라이브러리
- 제이쿼리
- AOS
- Bootstrap 아이콘
